# dysarthric-speech-recognition

This is a ML research project owned by Oren Schmerling and Walter Lloyd with the goal of training a neural network to understand and clarify dysarthric speech (creating a speech to speech model)

## Running the code

Set up your environment with `pipenv --python 3.11` and then `pipenv install`

to run the code, use `pipenv shell` then `python modelCompare.py --domain_a_dir < class A directory > --domain_b_dir < class B directory >`
